# repraesent.com v0.2.1 — Design Improvements

## 1. List of Improvements Made

### Design Tokens
- Updated color palette for premium, calm tone (warmer off-white, dark charcoal, muted olive)
- Added spacing scale tokens (--space-xs through --space-xl)
- Increased section gap from 6rem to 7.5rem (120px)
- Added --section-gap-mobile: 5rem

### Typography
- H1: clamp(2.25rem–3.25rem) → clamp(3rem–4.5rem), editorial headline feel
- H2: clamp(1.65rem–2rem) → clamp(1.85rem–2.35rem)
- H3: 1.2rem → 1.25rem
- Body: 1.05rem/1.65 → 1.0625rem/1.7 (improved readability)
- Hero text: 1.15rem → 1.125rem, max-width 38ch
- Reduced font-weight 600 → 500 across nav, buttons, service blocks, audience

### Hero
- Padding: 6rem/5rem → 7rem/6rem
- Hero title margin-bottom: 1.25rem → 1.75rem
- Hero text margin: 1.75rem → 2.25rem

### What's Included
- Converted from tight grid (gap 0.5rem) to vertical list (gap 1rem)
- Simplified bullet: em-dash → minimal dot
- Increased line-height to 1.6
- Included-note margin-top: 1.25rem → 1.5rem

### How It Works
- Flow diagram gap: 0.75rem/1rem → 1.5rem/2rem
- Font sizes: flow 1.25rem, items 1.2rem, arrows 1.1rem

### Pricing (High Priority)
- Bundle amount: 1.75rem/600 → 2.75rem/500, serif
- Bundle note: text-muted, 0.9rem
- Table: lighter typography (0.95rem), reduced padding
- Individual items: text-secondary for names
- Mobile: pricing amount 2.25rem

### Additional Services
- Removed border-bottom from service blocks
- Gap: 1.5rem → 2rem
- H3: font-weight 600 → 500, lighter feel

### Form
- Form-group margin: 1.25rem → 1.5rem
- Input padding: 0.6rem/0.75rem → 0.75rem/0.85rem

### FAQ (Reduce Noise)
- Removed full border and background from details
- Border-bottom only, transparent background
- Simplified container feel

### Responsive
- Section padding: 4rem → var(--section-gap-mobile) (5rem)
- Hero mobile: 4rem/3.5rem → 5rem/4rem
